BinListClient
=======================
A mobile client for binlist.net 

Task:
---------------------
* Create an app displaying card info using binlist.net
* Previous requests are displayed as a list
* Request history isn't cleared after restarting an app
* Phone, URL and coordinates are clickable and open appropriate apps to process data

Learned and worked with:
* Room
* Retrofit
* Dagger2
* Navigation
* Coroutines (+ StateFlow)
* ViewBinding
* Caching net requests
* Clean Architecture
* MVVM
* Implicit intents
